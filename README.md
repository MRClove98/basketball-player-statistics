# basketball-player-statistics
This is a basketball statistic collecter, where you can input player's name, points and amount of players. The program also calculates the highest score, the lowest scorer and the average points all of them scored. We made this program easy and fun to use!

## Features
-Input player's name
-Input amount of players
-Input points scored for each player
-Gives back report for highest scorer, lowest scorer and average points scored.
